SignTitles

oxguy3

View on GitHub

SignTitles is a Minecraft mod that allows you to add titles to signs with simple text shortcuts. As of v0.1, the only option is to put "[title]" on the line below the text you want to be vertically centered.

Changelog

v0.2 (in progress)

  • Support for parameters: adding parameters to the "[title]" tag will tweak how it appears
    • "l" displays lines above and below the title
    • "s" displays spaces between every character in the title (even if that means rendering beyond the sign's edges)

v0.1

  • Initial release
