SignTitles

oxguy3

SignTitles is a Minecraft mod that allows you to add titles to signs with simple text shortcuts. To add a title to a sign, simply put your title on one line and "[title]" on the line below it. See this image for more information.

To install SignTitles, download and extract the zip file, then copy the ".class" files into your Minecraft jar file (don't forget to delete META-INF in the jar file as well). You can download any version of the mod by clicking the version number in the changelog.
